Google Cloud 서비스 계정

① Google Sheets API 활성화

브라우저에서 console.cloud.google.com → 프로젝트 선택
APIs & Services → Enabled APIs
→ Google Sheets API 활성화 확인

② 서비스 계정 JSON 키 생성

IAM & Admin → Service Accounts → 계정 선택
Keys 탭 → ADD KEY → Create new key → JSON
다운로드된 .json 파일 저장 (다음 슬라이드에서 사용)
⚠️

API 키만으로는 Google Sheet에 쓸 수 없습니다.
서비스 계정 JSON이 반드시 필요하며,
시트에 편집자로 공유되어야 합니다.

💡 서비스 계정이란?

사람 대신 API를 호출하는 봇 계정
JSON 파일 안에 private key 포함
→ 절대 커밋/공유 금지

1

Chromium 브라우저가 자동으로 열립니다 AirDNA 서울 listings 페이지로 이동

증상 쓰기 실패, RESOURCE_EXHAUSTED
원인 API 쓰기 60회/분 초과
해결 자동 retry 내장 (20/40/80초)
반복 시 20~30분 후 재실행
